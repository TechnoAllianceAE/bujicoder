Slash Commands

Quick actions you can run from the chat input. Type / to see autocomplete suggestions.

All Commands

CommandDescription
/newStart a new conversation (clears chat history)
/mode [name]Switch cost mode. Without argument, shows a picker. Options: normal, heavy, max, plan
/historyBrowse and resume past conversations
/copyCopy the last assistant response to clipboard
/aboutShow version, Go runtime, and config paths
/initAnalyse project docs and explain the codebase
/modelsList available models and current mode-to-model mappings
/mcpList configured MCP servers and their status
/mcp add <name> <cmd> [args]Add a new MCP server (lazy by default)
/mcp remove <name>Remove an MCP server from config
/memoryView project memory (BUJI.md)
/goal <description>Decompose a goal into tasks and execute with specialist agents (details)
/refreshReload config and refresh model-agent assignments
/updateCheck for BujiCoder updates
/helpShow help and keyboard shortcuts
/quitExit BujiCoder
/exitExit BujiCoder (alias for /quit)

Autocomplete

When you type / followed by characters, BujiCoder shows a dropdown of matching commands. Use:

  • Up / Down to navigate the list
  • Tab or Right to accept the selection
  • Esc to dismiss

File Attachments

Attach files to your message using the @ prefix:

@path/to/file.py Can you explain this code?
@screenshot.png What's wrong with this UI?

BujiCoder supports text files and images (PNG, JPG, etc.) for multimodal models.
