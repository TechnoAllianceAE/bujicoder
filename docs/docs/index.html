What is BujiCoder?

BujiCoder is a Go-native, multi-agent AI coding assistant that runs in your terminal. It ships as a single binary with zero runtime dependencies — no Node.js, no npm, no Python.

You bring your own API keys from any supported LLM provider and BujiCoder routes your requests directly. Your code never leaves your machine.

Quick Start

Get up and running in under a minute:

1. Install

curl -fsSL https://bujicoder.com/install.sh | bash

2. Launch

buji

The setup wizard runs automatically on first launch. It asks you to pick a provider and enter an API key.

3. Code

Start typing natural-language instructions. BujiCoder reads files, edits code, runs terminal commands, searches your codebase, and commits changes — all from the chat interface.

Tip: Press Shift+Tab to cycle between cost modes (normal / heavy / max / plan) without leaving the chat.
