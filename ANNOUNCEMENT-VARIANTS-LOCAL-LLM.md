**BujiCoder v0.x: Local LLM Support (Ollama & llama.cpp)**

**Summary**

Added support for locally-hosted LLMs through Ollama and llama.cpp. Enables
fully offline, private AI-assisted coding without cloud dependencies.

**Changes**

1. **New LLM Provider: llamacpp**
   - Added `shared/llm/llamacpp.go` provider wrapper
   - Supports llama.cpp server protocol (OpenAI-compatible)
   - Environment variable: `LLAMACPP_URL`

2. **Improved Setup Wizard**
   - Added Ollama and llama.cpp options to first-run setup
   - URL config flow for local model servers
   - Fallback default mappings

3. **Config & Env Var Support**
   - Added `LLAMACPP_URL` support
   - Unified config structure across all providers
   - Support for local model tag/model list fetching

4. **Error Handling Improvements**
   - Structured error types with rate-limit info
   - Better error propagation across providers
   - Fixed 'NormalizeHeaders' binding issues

**Files Changed**

cli/app/model.go            — Register llamacpp provider
